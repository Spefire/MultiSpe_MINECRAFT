# MultiSpe
RPG plugin designed to add a new combat experience to Minecraft.

Each player can choose a class:
- Warrior
- Archer
- Priest
- Wizard
- Druid
- Necromancer

And gain access to unique abilities divided into several categories:
- [ATK] — offensive skills: powerful strikes, poison, magic arrows, explosions...
- [DEF] — defensive abilities: invisibility, rage, berserker mode, enhanced mobility...
- [ZONE] — area skills: temporal prison, fire, implosion, crowd control...

The plugin also includes:
- ✔️ Cooldown and recharge management
- ✔️ Immersive visual effects, particles, and sounds
- ✔️ Permission system
- ✔️ Multi-language support (EN + FR)~~~~

MultiSpe allows you to create adventure / RPG / semi-RP servers while remaining easy to use.

## Usage

### Basic Commands

- **/ms** – Show plugin help
- **/msStatus** – Show your current class
- **/msClass <Warrior|Archer|Priest|Wizard|Necromancer>** – Choose a class
- **/msUnclass** – Remove your class
- **/msStuff** – Get your class equipment
- **/msLanguage <en|fr>** – Change plugin language
- **/msLocation** – Check if the plugin is active at your location
- **/msSetWorld <true|false>** – Enable / disable plugin in the current world

### Permissions

- **multispe.admin** – Can use all commands *(default: op)*
- **multispe.add.class** – Can choose a class if you don’t have one *(default: false)*
- **multispe.change.class** – Can change to another class *(default: false)*
- **multispe.remove.class** – Can remove your class *(default: false)*
- **multispe.class.*** – Can select all classes *(default: false)*
- **multispe.class.war** – Can select Warrior *(default: false)*
- **multispe.class.arc** – Can select Archer *(default: false)*
- **multispe.class.pri** – Can select Priest *(default: false)*
- **multispe.class.wiz** – Can select Wizard *(default: false)*
- **multispe.class.dru** – Can select Druid *(default: false)*
- **multispe.class.nec** – Can select Necromancer *(default: false)*
- **multispe.stuff** – Can receive class equipment *(default: false)*
- **multispe.setworld** – Can enable/disable plugin in worlds *(default: op)*
